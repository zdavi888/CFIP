# Cloudflare WARP 自动优选多路 SOCKS5 本地代理池工具
## 基于 Xray-core 高性能多路分流架构

本工具是一个专为指纹浏览器、多开养号、分布式抓取而设计的多通道高配 Proxy 代理池。能在本地一键组装出 **30 个独立的 SOCKS5 通道**（10801 ~ 10830），每个通道直连一个 Cloudflare 官方专属注册的私有 WARP 链路，完全隔离，独立出口 IP，无限流量。

## 🌟 核心优势 (Xray-core 架构)

1. **绝对不与本地 V2Ray 冲突**：
   Xray 与 V2Ray 只是不同的客户端进程。只要它们占用的本地端口（如本工具的 10801~10830，以及您 V2Ray 的 10808 等）不发生重叠，就可以同时完美并发运行，完全互不干涉！
2. **免去本地 SmartScreen 或杀毒拦截困扰**：
   我们的云端打包系统（Build-Time）已经为您**预先下载并解压**了完美匹配的 Windows 64位安全核心 `xray.exe`，免去了本地打包、PowerShell 阻断、无法下载的各种报错！
3. **极高稳定性，防升级语法作废**：
   相比 Sing-Box 经常由于微小版本升级导致配置文件字段如 `server` / `sniffer` 作废，Xray-core 使用标准的、跨版本高度兼容的 Xray/V2Ray 规范，持久可靠。

## 🚀 极速启用步骤

### 【第一步：下载并解压此文件夹】
直接在本平台上点击 **ZIP 导出**，解压到您的本地电脑（例如桌面）。

### 【第二步：一键双击启动】
* **Windows 系统**：直接双击文件夹内的 `start.bat`。
* **Linux/macOS 系统**：在终端运行 `./start.sh`。

工具会自动进行以下极其智能的操作：
1. **[1/3] Cloudflare 物理测速**：自动在多组 Cloudflare 官方 Anycast 集群 IP 段中测试延迟，精选出前 30 个对您本地网络延迟最短的物理中继节点。
2. **[2/3] 官方独占凭证分配**：在本地为每个端口分别发起安全的算法注册，向 Cloudflare 申请独立的 WARP 物理隔离密匙和 Client ID，免去共用被封锁的危险。
3. **[3/3] 自动生成配置并飞速秒开**：自动生成 `config.json`，并由内置的 Xray-core 运行起高并发代理池！

## ℹ️ 指纹浏览器配置说明

一旦在黑窗口看到 `恭喜您！所有本地多路代理链路已就绪` 提示后，即可在指纹浏览器中进行多开并独立关联：

* **代理类型**：`Socks5`
* **代理主机(IP)**：`127.0.0.1`
* **代理端口**：输入 `10801` 到 `10830` 中的任意一个端口。
* 如需多开 30 个浏览器，只需分别填写 `10801`、`10802` 直到 `10830`。每个浏览器的物理路由、出口公网 IP 全部相互独立，达到完美的防关联效果！

## 🌐 VPS 集中公网部署与指纹浏览器直接调用指南

如果你想要开一个 **VPS（云服务器）** 专门用来生成、承载、并且输出这 30 个出口 IP 的 Socks5 代理，让你的**本地指纹浏览器**能跨越网络直接连接：

### 1. 安全前置知识（为什么不能裸连 127.0.0.1）
如果将 SOCKS5 代理无密码公开在 VPS 公网上，会立即引来黑客扫描并被盗用带宽。因此，我们专门在 `start.js` 中开发了 **「安全高防护墙（SOCKS5 密码认证模型）」**！

### 2. 在 VPS 上极速配置并起航：
1. **生成并获取带密码的配置**：直接在前台 Web 网页上勾选 **「VPS 部署与远程连接安全配置」**，设置你自定义的账号与密码（比如 `admin` 和 `mysecurepwd999`）。网页上的 Xray-core 配置文件会自动变更为监听 `0.0.0.0` 且带有密码盾的配置。
2. **在 `start.js` 中锁定密码**：打开 `/获取CFIP/start.js`，修改开头的两个变量：
   ```javascript
   const SOCKS5_USER = 'admin';       // 您刚才设置的用户名
   const SOCKS5_PASS = 'mysecurepwd999'; // 您刚才设置的密码
   ```
3. **把代码上传到你的 VPS 并在 VPS 运行**：
   * 在您的 VPS 上安装 Node.js 和 unzip 工具。
   * 运行启动命令：`node start.js 30` (30代表开通30个通道)。
   * 程序探测最优质 Anycast、申请官方密匙，并开启高防 Socks5。

### 3. 本地指纹浏览器直接取 IP 和端口连接：
在本地的电脑（如 AdsPower、比特、Hubstudio 等指纹浏览器）中这样配置：
* **代理协议**：`Socks5`
* **代理主机 (IP)**：`【填写您的 VPS 公网 IP】`
* **代理端口**：`10801` 到 `10830`
* **登录用户**：`admin`
* **登录密码**：`mysecurepwd999`

**🚀 极速物理网络链路**：
`本地指纹浏览器` ==> `通过中继直达到 VPS (通过高防护墙安全握手)` ==> `VPS 内部通过 30 组隔离 Anycast 通道出网` ==> `CF WARP 数据中心 (伪装为指纹独享住宅IP)` ==> `目标网站`

这套方案 100% 杜绝了防火墙任何可能的干扰，性能强悍，且 VPS 只需轻量配置即可完美支撑 (内存占用极其节省)！
